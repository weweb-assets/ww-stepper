# ww-stepper

Stepper element to wrap multi-step forms. Each step is a droppable content area; a header renders numbered or bulleted step indicators; optional navigation buttons move between steps.

- Styles: step style (Numbers / Bullets), completed-step icon, icon color, active / inactive / current colors, number font size.
- Settings: steps (repeatable dropzones), orientation (horizontal / vertical), clickable steps, show labels, show navigation buttons.
- Triggers: `On step change`, `On step complete`, `On finish`.
- Component actions: `Go to step`, `Next step`, `Previous step`, `Trigger finish` (+ editor step management).
- Variables: `activeStepIndex`, `completedSteps`, `isFirstStep`, `isLastStep`.
